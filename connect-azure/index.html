    Connect Azure

    For Coster to get access to your Azure subscription(s), you need to

    1. Register an application in your Azure AD;
    2. Assign Contributor role to this application;

    3. Gather and share the following information with Coster:

      • Tenant ID,
      • Application (client) ID,
      • Client secret.

    Note

    Action requires Owner or User Access Administrator role.

    How to get Tenant ID

    Sign in to your Azure Account through the Azure portal. Select Azure Active Directory. Copy Tenant ID.

    How to register a service client application

    1. On the Azure Active Directory blade select App registrations.
    2. Select New registration. Name the application, e.g. Coster-app.
    3. Select a supported account type, we recommend to use default settings. After setting the values, select Register. Copy application data and insert to fields.

    How to get application (client) secret

    1. In the App registration blade select application, e.g. Coster-app.
    2. Select Certificates & secrets.
    3. Select New client secret.
    4. Provide a description and duration of the secret (either 1 year, 2 years or never).
    5. Once the secret has been generated, it will only be displayed once in the portal. Make a note of it and store it securely.

    How to assign role to the application

    Coster gathers the data which is available to the Azure AD application. To assign role in a single subscription do the following steps.

    1. Select the particular subscription to assign the application to.
    2. Select Access control (IAM).
    3. Select Add > Add role assignment to open the Add role assignment page.
    4. Select Contributor role to assign to the application. Click Next.
    5. Select who needs access. On the Members tab, select User, group, or service principal. Click Select members.
    6. Type in the Select box the name of the application, e.g. Coster-app. Click Select.
    7. Click Review + assign to assign the role.

    To assign role in several subscriptions implement one of the options:

    a. Add application to Active Directory group of the Security type, group should have a role of at least Contributor and the access to the necessary subscriptions. In this case you do not need to assign role to the application specifically. More on AD groups.

    b. Add application to the service Management groups:

    i. In the upper search field enter "Management groups". Click the respective service in the search results.

    ii. From the list of your management groups, choose the one that has access to the necessary subscriptions.

    iii. Select Access control (IAM). Follow the same steps as for assigning role in a single subscription. More on management groups.

    Get list of subscriptions

    After you enter Tenant ID, Application ID and Client secret click Get subscriptions. If connection is successful, drop-down menu near the button will list available subscriptions. Choose subscriptions you want to connect to Coster.

    Check connection

    Note

    Depending on the amount of data upload can take up to 10 min.

    Open Budgets. Click Create budget.

    In budget scope, on the resource tree you should be able to find subscriptions and resource groups.

    Data is synchronized daily at 10:00 UTC.
